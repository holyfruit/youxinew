# **App Name**: Stickman Champion

## Core Features:

- Level Creation: Design various challenging levels with obstacles and enemies.
- Stickman Animation: Animate the stickman character with fluid movements for running, jumping, and fighting.
- Physics Engine: Implement a physics engine to simulate realistic movements and collisions.
- Enemy AI: Create basic AI for enemies to patrol, chase, and attack the player. Utilize AI to choose enemy behavior dynamically.
- Victory Message Customization: The LLM generate a custom victory message for Chen Zhiyan using the hardcoded instructions provided to encourage learning.
- Score Tracking: Keep track of the player's score and display it on the screen.

## Style Guidelines:

- Primary color: Dark gray (#333333) for the stickman and level elements.
- Background color: Light gray (#F0F0F0) to provide contrast and readability.
- Accent color: Bright red (#FF4500) for interactive elements and highlights.
- Font: 'Inter', a sans-serif, for a clean and modern look in the UI.
- Simple, geometric icons for in-game UI elements like score and pause.
- Clean and minimalistic layout with a focus on gameplay area.
- Smooth and responsive animations for character movements and interactions.
